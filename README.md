# IBM – Generative AI Apps

Raccolta di progetti sviluppati durante il corso **“Building Generative AI‑Powered Applications with Python” (IBM, Coursera)**.  
Ogni cartella contiene una piccola applicazione che usa modelli di Generative AI in modo pratico.

## Progetti

### 1. ChatGPT‑like Web App (`mod2-chatbot-website`)
Semplice chatbot web che usa un modello LLM open‑source / API per:
- rispondere a domande degli utenti,
- mantenere il contesto della conversazione,
- mostrare la cronologia chat in un’interfaccia web.

Tecnologie principali: Python, framework web del corso (es. Gradio/Flask), API LLM.

### 2. Voice Assistant (`mod3-voice-assistant`)
Assistente vocale che:
- converte la voce in testo (STT),
- usa un LLM per generare la risposta,
- legge la risposta ad alta voce (TTS).

Tecnologie principali: Python, servizi STT/TTS IBM / provider cloud, API LLM.

### 3. Private Data Summarizer (`mod5-private-data-summarizer`)
App che permette di caricare documenti/testi privati e ottenere:
- riassunti,
- punti chiave,
- risposte a domande sul contenuto.

Tecnologie principali: Python, LLM, elaborazione file.

*(Altri progetti verranno aggiunti man mano che completo i moduli successivi.)*
